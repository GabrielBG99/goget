# goget

A download accelerator written in Go

## Installation
```bash
git clone https://github.com/GabrielBG99/goget
cd goget
make install
```

## Usage
Use `gogoet help` to see the list of commands

- Exemple:
  ```bash
  goget single -n 10 -u https://golang.org/lib/godoc/images/go-logo-blue.svg
  ```
